CSS Basics

CSS is used to style an HTML document


  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

Basic Properties and Units with Example

Units in CSS (px)

Box of One Hundred Pixels (height and length): 100px

id = "box1"


Units in CSS (%)

Box of Fifty Percent (height and length): 50%

id = "box2"



This box is unable to compute the height because the body has no fixed height. It is able to compute the width because it is equal to viewport. Let us make a box inside a parent with a fixed height


"Outer Box" of Two Hundred Pixels (height and length): 200px

"Inner Box" of 50% (height and length): 50%

id = "box3"


Units in CSS (rem & em)

Box of 4rem (height and length): 4rem

id = "box4"



rem - is a relative unit that refers the font of root element. In our case 1rem = 16px, therefore 4rem = 64px. The font size of the root element can be affected by user's preference in the browser settings.


"Outer Box" of Two Hundred Pixels (height and length): 200px, and font size of 12px

"Inner Box" of 4em (height and length): 4em

id = "box5"


em - is another relative but with one difference, if applied to a child element it will refer to the font size of parent element. In our case the inner box will have the following size, 4em = (12 * 4)px = 48px
